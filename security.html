Security

Security is crucial for any commerce site. Below are recommended measures and a brief explanation for each.

Recommended practices

  • Use HTTPS with HSTS to protect data in transit.
  • Secure cookies and set proper SameSite attributes.
  • Hash and salt passwords (use strong algorithms like bcrypt/argon2).
  • Perform regular dependency and security updates and audits.
  • Use least-privilege access for services and store secrets securely (e.g., secret manager).

Incident response

Have a plan for security incidents: triage, notify affected users, and remediate root causes. For demos this is informational only.
